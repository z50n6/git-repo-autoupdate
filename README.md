



主要工具在[git-repo-autoupdate](https://github.com/z50n6/git-repo-autoupdate/blob/main/git-repo-autoupdate/README.md)路径下，可以自行打包使用，config.json文件可以自行添加想要定期更新的本地仓库路径。
